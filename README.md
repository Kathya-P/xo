# ❌⭕ Juego XO (Tic Tac Toe)

Un juego clásico de tres en raya desarrollado con JavaScript puro, HTML y CSS. ¡Simple, dinámico y muy divertido! 🎮

## 🎯 Descripción

Este es un juego de XO (también conocido como Tic Tac Toe o Tres en Raya) completamente funcional y fácil de jugar. Desarrollado con tecnologías web básicas, es perfecto para pasar un buen rato desafiando a un amigo o practicar tus habilidades de programación.

## ✨ Características

- 🎨 **Diseño atractivo y moderno**: Interfaz visual súper dinámica y amigable
- 🕹️ **Juego para 2 jugadores**: Modo multijugador local (X y O)
- ⚡ **Súper rápido**: Sin retrasos, respuesta inmediata
- 🏆 **Detección automática de ganador**: El juego identifica automáticamente al ganador o empate
- 🔄 **Botón de reinicio**: Comienza una nueva partida cuando quieras
- 📱 **Responsive**: Se adapta perfectamente a cualquier dispositivo
- 🎯 **Fácil de jugar**: Interfaz intuitiva, solo haz clic y juega

## 🛠️ Tecnologías

- **HTML5**: Estructura del juego
- **CSS3**: Estilos y animaciones
- **JavaScript (Vanilla)**: Lógica del juego

## 🚀 Cómo Jugar

1. **Abre el archivo `index.html`** en tu navegador favorito
2. **El jugador X siempre comienza** primero
3. **Haz clic en cualquier casilla** vacía para colocar tu símbolo
4. **Los jugadores se turnan** colocando X y O
5. **Gana el primero** que logre tres en línea (horizontal, vertical o diagonal)
6. **Si se llenan todas las casillas** sin ganador, es empate
7. **Presiona "Reiniciar"** para jugar otra partida

## 📦 Instalación

### Opción 1: Descarga Directa
1. Descarga todos los archivos del repositorio
2. Abre `index.html` en tu navegador
3. ¡A jugar! 🎮

### Opción 2: Clonar Repositorio
```bash
git clone https://github.com/m0nge/XO.git
cd XO
```
Luego abre `index.html` en tu navegador.

### Opción 3: Live Server (Recomendado para desarrollo)
Si tienes VS Code:
1. Instala la extensión "Live Server"
2. Click derecho en `index.html`
3. Selecciona "Open with Live Server"

Puedes personalizar fácilmente los colores y estilos editando el archivo `estiloXO.css`:

```css

:root {
  --color-x: #FF6B6B;
  --color-o: #4ECDC4;
  --color-fondo: #1A1A2E;
}
```
